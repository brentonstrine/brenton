Someone showed me how to save a text file with the html extension.

In High School, a teacher said she was going to put together a "coding class" and we were going to learn how to make websites. We only had one meeting, but in that one meeting, she showed me how to open a file with Notepad, save it with the `html` extension, and load it into a browser. I was amazed that with any computer, I had the power to publish something to the internet!

High School Self Learning

In High School, I read tutorials online and built websites in my spare time. I didn't have internet at home, so I used the school library computers in between classes. I would search for "learn HTML" and then find tutorials online, which I would save onto a floppy disc, take home with me to complete on my computer at home. As I was building sites for fun, I frequently saw websites doing things that I wanted to learn how to do, so I would view the sorce and try to figure out how they did it. That is how I picked up CSS and Javascript for the first time. Eventually, I went through tutorials on Javascript as well and developed a rudimentary understanding of the Javascript basics.

At the time, the easist way to put your website online for free was to use Geocities, or other free hosting websites. All of my earliest websites were hosting with Geocities.

School Newspaper Webmaster

I was hired to build the website for the school newspaper. I had to not only design and build the website, but I had to transfer content from the newspaper files to the website every week, and back up the old weeks to an archive. This was a huge pain, so I wrote a PHP script that would automatically build the site out based on me copy/pasting the content in. That way, each week, all I had to do was copy and paste the information into the script, save some images, and run the "build" operation. The result was a finished website that I uploaded to overwrite the old one.

  • PHP templating
  • Build and deploy processes
